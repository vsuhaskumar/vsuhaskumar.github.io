Secure Web Based Polling System Using Homomorphic Encryption Technique

Platform: HTML, CSS and JSP for frontend, My SQL for backend.
Tool: Eclipse IDE, MY SQL 5.5 and SQL YOG
Description: A web-based implementation done for standalone and distributed system. In traditional method, every vote must be decrypted to find the result. This results in time consumption. This project aims at maintaining the confidentiality of voters and reducing the time required for decryption. To achieve this, we use Homomorphic Encryption technique. Here we do not decrypt every vote, rather we decrypt total sum of votes and find the results.
